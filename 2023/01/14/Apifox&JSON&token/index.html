JSON

是什么：一种可以在服务器和客户端之间传输的数据格式，非编程语言
json在线检查语法：https://www.json.cn/
基本语法

  • 数组（Array）用方括号(“[]”)表示。
  • 对象（0bject）用大括号(“{}”)表示。
  • 名称/值对(name/value）组合成数组和对象。
  • 名称(name）置于双引号中，值（value）有字符串、数值、布尔值、null、对象和数组。
  • 并列的数据之间用逗号(“,”）分隔

Json与js对象互转

  • **Json.parse **json ——> js对象
  • **Json.stringify **js对象 ——> json

token

“令牌”，是服务器生成的一段加密字符串，然后返回给客户端
当第一次登录后,服务器生成一个token便将此token返回给客户端,以后客户端只要带上这个token前来请求数据即可,无需再次带上用户名和密码
基于token的身份验证：
客户端在初次登录时，填写好的账号密码发送到服务端，然后服务端对此进行验证。
验证成功后，服务端签发一个token，之后把这个token发送给客户端。
客户端将这个token进行存储，比如放到local storage或cookie中，此后需要向服务器请求某些资源时就携带这个token，服务端再对token进行验证。
验证成功返回有效信息，错误返回状态码。
需要注意，token是具备有效期的，有时候显示登录过期代表token过期，这时就需要重新登录验证。
token的使用：
登录时返回token

请求数据时，token通常作为请求参数
